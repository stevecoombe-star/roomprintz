Drop your own room images here for local/admin 3D Room Lab testing.

Usage:
- Put image files in this folder: public/3d-lab/room-images/
- Reference them in the lab's "Room image URL" field as:
    /3d-lab/room-images/<filename>
- Example:
    /3d-lab/room-images/steve-living-room.jpg

Notes:
- Common web formats work (jpg, png, webp).
- These are local test assets only; do not commit private/customer photos.
- This README keeps the folder tracked by Git even when empty.
